# Privacy Boundary

This repository should remain safe as a scaffold even if it is later shared selectively.

## Do Not Commit

- real AI chat logs;
- recordings or transcripts;
- private session IDs;
- tool-specific session URIs or internal conversation links;
- personal themes, projects, outputs, or memories;
- `.obsidian/` workspace state;
- local absolute paths such as `/Users/name/...`;
- secrets, credentials, API keys, tokens, private keys, or internal URLs.

## Allowed In This Repository

- protocols;
- templates;
- empty directory scaffolds;
- fictional examples explicitly named `example-*`;
- generic launcher prompts and tool-native skill packages with placeholders such as `<VAULT_PATH>`.

## If You Turn This Into A Real Private Vault

Either fork it privately or relax `.gitignore` intentionally. Do not casually commit real data into a repository that may later be shared.
