# 10_sources

## What belongs here

Raw evidence, excerpts, files, links, transcripts, AI chat pointers, recordings, papers, documents, screenshots, and codebase observations.

## What does not belong here

Durable memory, project state, theme maps, or polished reusable outputs.

## When to write here

Write here first when preserving evidence or a stable pointer for future reasoning.

## Who reads it

AI agents read sources before writing sessions, themes, projects, memories, or briefings. Humans read sources when they need provenance.

## Common mistakes

- Treating source notes as long-term memory.
- Copying entire repositories, large media libraries, or private datasets into the vault.
- Omitting provenance such as URL, file path, captured time, or tool session id.
