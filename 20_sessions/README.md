# 20_sessions

## What belongs here

Summaries of meaningful conversations, readings, meetings, work sessions, or historical AI tool sessions.

## What does not belong here

Raw evidence, confirmed memory, or full project plans that belong in `50_projects/`.

## When to write here

Write a session summary when future work needs continuity.

## Who reads it

AI agents use sessions to resume context, update themes/projects, propose candidate memories, and create briefings.

## Common mistakes

- Writing conclusions without linking source evidence.
- Losing topic shifts in long conversations.
- Treating session conclusions as permanent truth.
