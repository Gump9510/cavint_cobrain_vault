# 40_themes

## What belongs here

Long-running domains of inquiry, evolving cognitive maps, principles, mechanisms, contradictions, and open questions.

## What does not belong here

Project execution state, raw evidence, or confirmed memory records.

## When to write here

Update a theme when understanding of a long-running domain changes or multiple sessions/projects point to the same domain.

## Who reads it

AI agents read themes to understand long-term context and judgment frameworks.

## Common mistakes

- Turning a theme into a project task list.
- Writing unsourced conclusions.
- Forgetting to link sessions, sources, projects, memories, or outputs.
