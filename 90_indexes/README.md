# 90_indexes

## What belongs here

Indexes, templates, registries, retrieval briefings, change logs, and routing notes.

## What does not belong here

Raw evidence, durable memory bodies, or primary project/theme content.

## When to write here

Write here when the system needs navigation, reverse lookup, templates, continuity, or audit trails.

## Who reads it

AI agents read indexes to start correctly and verify relationships. Humans read them to find things and understand recent changes.

## Common mistakes

- Treating indexes as the knowledge body.
- Updating content without updating registry or change log when required.
